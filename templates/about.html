About This Project

Feng Shui Your Room!

This is an educational web application designed to teach the principles of Feng Shui through interactive learning and a room simulation experience.

The application includes:

  • Educational content on the history and concepts of Feng Shui
  • An interactive room simulator to practice furniture placement
  • A quiz to test your knowledge
Learning Objectives
  • Understand the history and philosophy of Feng Shui
  • Apply interior layout rules based on qi flow
  • Reinforce learning through practical simulation
  • Build intuitive understanding rather than rote memorization
Technical Implementation

This prototype was built using:

  • Flask (Python web framework)
  • Jinja2 (Templating engine)
  • Bootstrap (UI Framework)
  • jQuery (JavaScript library)
  • jQuery UI (For drag-and-drop functionality)
