About

Feng Shui Your Room is an educational web application designed to teach the principles of Feng Shui through interactive learning and a room simulation experience.

The application includes:

  • Educational content on the history and concepts of Feng Shui
  • An interactive room simulator to practice furniture placement
  • A quiz to test your knowledge

The primary learning objectives are:

  • Understanding basic history and philosophy of Feng Shui
  • Applying interior layout rules based on qi flow, primarily in a bedroom setting
  • Reinforcing learning through hands-on simulation
  • Building intuitive understanding rather than rote memorization

This project was done by Jennifer Fu and Junhong Liao for User Interface Design (COMS4170) at Columbia University. The site was built using the following libraries and frameworks:

  • Flask
  • Jinja2
  • Bootstrap
  • jQuery
  • jQuery UI (drag-and-drop functionality)
